# Planning Templates

Copy-paste templates for the 3-file planning pattern.

## task_plan.md Template

```markdown
# Task: [Task Name]

## Goal
[Brief description of what needs to be accomplished]

## Context
[Any relevant background information, constraints, or requirements]

## Phases

- [ ] Phase 1: [Phase Name]
  - Description: [What this phase involves]
  - Success Criteria: [How to know it's complete]

- [ ] Phase 2: [Phase Name]
  - Description: [What this phase involves]
  - Success Criteria: [How to know it's complete]

- [ ] Phase 3: [Phase Name]
  - Description: [What this phase involves]
  - Success Criteria: [How to know it's complete]

## Current Status
**Phase**: [Current phase number]  
**Progress**: [Brief status update]  
**Blockers**: [Any current blockers or issues]

## Notes
[Any additional notes or considerations]
```

## findings.md Template

```markdown
# Findings

## Overview
[Brief summary of research topic]

## Key Discoveries

### [Topic 1]
- **Source**: [Where this info came from]
- **Finding**: [What was discovered]
- **Relevance**: [Why this matters for the task]

### [Topic 2]
- **Source**: [Where this info came from]
- **Finding**: [What was discovered]
- **Relevance**: [Why this matters for the task]

## Code Analysis

### [File/Module Name]
- **Purpose**: [What this code does]
- **Usage**: [How to use it]
- **Notes**: [Any important details]

## API Documentation

### [API/Library Name]
- **Endpoint/Method**: [Name]
- **Parameters**: [List parameters]
- **Returns**: [Return type/value]

## Architecture Notes
[Any architectural decisions, patterns, or structures discovered]

## References
- [Link 1]: [Description]
- [Link 2]: [Description]
```

## progress.md Template

```markdown
# Progress Log

## Session: [Date/Time]

### Actions Taken

#### [Timestamp] [Action Name]
- **Command**: [command if applicable]
- **Result**: [What happened]

### Tests Run

#### [Timestamp] [Test Name]
- **Command**: [test command]
- **Status**: Pass or Fail

### Error Log

#### [Timestamp] [Error Name]
- **Attempt**: [1/2/3]
- **Error**: [Description of what failed]
- **Context**: [What was being attempted]
- **Next Step**: [What will be tried next]
```
