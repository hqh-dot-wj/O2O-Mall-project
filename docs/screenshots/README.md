## 截图建议

1. **清晰度**：确保截图清晰，文字可读
2. **完整性**：包含完整的页面布局（头部、侧边栏、内容区）
3. **数据**：使用真实或模拟的数据，避免空白页面
4. **浏览器**：建议使用 Chrome 浏览器无痕模式截图
5. **分辨率**：推荐 1920x1080 或 1440x900
6. **格式**：PNG 格式，保证图片质量

## 截图工具推荐

- macOS: 自带截图工具 (Shift + Command + 4)
- Windows: Snipping Tool / Snip & Sketch
- Chrome 扩展: Full Page Screen Capture
- 专业工具: Snagit、FastStone Capture
