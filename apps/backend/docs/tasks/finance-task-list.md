## 开发顺序说明

按模块依赖关系排序：**Wallet → Commission → Settlement → Withdrawal**

- Wallet 是基础层，所有资金变动最终都要操作钱包
- Commission 依赖 Wallet 进行佣金回滚
- Settlement 依赖 Commission + Wallet
- Withdrawal 依赖 Wallet
