## 📊 功能特性

### 数据统计维度

```
分销数据看板
├── 分销员统计
│   ├── 总分销员数（levelId >= 1）
│   ├── 新增分销员数（时间范围内）
│   └── 活跃分销员数（有佣金记录）
├── 订单统计
│   ├── 分销订单总数
│   ├── 分销订单总金额
│   └── 分销订单占比（%）
└── 佣金统计
    ├── 佣金支出总额
    ├── 待结算佣金（FROZEN）
    ├── 已结算佣金（SETTLED）
    └── 佣金趋势（按日）
```

### 查询优化

1. **并行查询**：使用 `Promise.all` 并行查询三个统计维度
2. **聚合查询**：使用 `aggregate` 和 `groupBy` 减少数据传输
3. **默认时间范围**：默认查询最近30天，避免全表扫描

### 数据处理

- 金额类型转换：`Decimal` → `number`
- 日期格式化：`Date` → `YYYY-MM-DD`
- 百分比计算：保留2位小数
- 空值处理：`null` → `0`
